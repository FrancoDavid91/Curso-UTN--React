Dada la siguiente función en JavaScript:

        
            function suma(a, b, callback) {
                let c = a + b;
                callback();
            }
  1. ¿Qué es el tercer parámetro recibido?

    2. El tercer parámetro recibido en la función "suma" es la función "callback". En este contexto, una función de "callback" es una función que se pasa como argumento a otra función y se ejecuta después de que se complete cierta tarea.

  2. ¿En qué casos es obligatorio desarrollar este tipo de funciones?

    3. Las funciones de "callback" son necesarias en programación para gestionar operaciones asíncronas y personalización de comportamiento. Se utilizan en situaciones en las que es importante controlar cuándo y cómo se ejecutan ciertas tareas.

  3. Realizar un llamado a la función de ejemplo.
